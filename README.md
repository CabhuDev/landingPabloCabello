## 🛠️ **Instalación y Desarrollo**

### **🚀 Desarrollo Local:**

#### **Opción 1: Scripts automatizados (recomendado)**
```bash
# Windows (PowerShell)
./scripts/dev-server.ps1

# Docker local (Windows)
./scripts/docker-local.ps1
```

Los scripts automáticamente:
- ✅ Verifican dependencias (Node.js, Python)
- ✅ Instalan `serve` si falta  
- ✅ Inician backend (FastAPI) y frontend (serve)
- ✅ Abren navegador automáticamente

#### **Opción 2: Manual**
```bash
# Terminal 1 - Backend
cd backend
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

# Terminal 2 - Frontend  
cd frontend
npx serve . --cors
```

### **🐳 Despliegue con Docker**

#### **Construcción:**
```bash
./scripts/docker-build.sh [tag]
```

#### **Desarrollo con Docker:**
```bash
docker-compose up -d
```

#### **Producción:**
```bash
./scripts/docker-deploy.sh
```

> 📖 **Documentación completa Docker:** [docs/README-Docker.md](docs/README-Docker.md)

### **📊 Configuración de Métricas:**

#### **Google Analytics 4:**
1. Crear propiedad en `analytics.google.com`
2. Copiar Tracking ID (`G-XXXXXXXXXX`)
3. Reemplazar en `index.html` línea 101

#### **Hotjar (Heatmaps):**
1. Crear cuenta en `hotjar.com`
2. Copiar Site ID
3. Reemplazar en `index.html` línea 146

#### **A/B Testing:**
```javascript
// Activar en producción:
localStorage.setItem('enable_ab_testing', 'true');
```

Ver guía completa en → **[docs/METRICAS-IMPLEMENTACION.md](docs/METRICAS-IMPLEMENTACION.md)**
